# The Last Christmas 
A short atmospheric horror game developed in 3 days for a Game Jam.

## About the Game 

The Last Christmas is a short atmospheric horror experience created during a Game Jam.

The project focuses on atmospheric horror. 

## Built With 

- Godot Engine 4.3

- GDScript
